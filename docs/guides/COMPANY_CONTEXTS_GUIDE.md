# Company Contexts Extraction Guide

## Overview

The `build_company_contexts.py` script extracts company mentions from news history using spaCy NER, normalizes company names to canonical forms, and generates context snippets for enrichment purposes.

## Purpose

This script creates a foundation for company data enrichment by:
1. Identifying all company mentions across historical articles
2. Normalizing company name variations to canonical forms
3. Collecting context snippets showing how companies are discussed
4. Filtering out noise entities (generic terms, media outlets, venues)

## How It Works

### 1. Data Loading

Reads `data/news_history.csv` and combines `title` + `summary` as article text.

### 2. NER Processing

Uses spaCy's `en_core_web_sm` model to extract ORG (organization) entities from each article.

### 3. Name Normalization

**Algorithm:**
```python
def canonical_company_name(raw: str) -> str:
    """
    Examples:
        "Flutter Entertainment plc" -> "Flutter Entertainment"
        "DraftKings Inc." -> "Draftkings"
        "Entain PLC" -> "Entain"
    """
    1. Lowercase the name
    2. Remove punctuation
    3. Drop legal suffix tokens: plc, ltd, inc, corp, etc.
    4. Keep first 2-3 remaining tokens
    5. Return title-cased
```

**Legal Suffixes Removed:**
- plc, ltd, limited
- inc, corp, corporation
- group, holdings, company
- nv, sa, se, ag, llc, llp

### 4. Noise Filtering

**Two-tier filtering:**

**Tier 1: Taxonomy Integration**
- Imports ignore list from `taxonomy.py` if available
- Filters out generic terms, sources, internal brands

**Tier 2: Context-Specific**
- Additional entities specific to company extraction:
  - Gaming terms: Slots, Poker, Blackjack, Roulette
  - Event venues: Barcelona, London, Amsterdam, Las Vegas
  - Media/Regulators: ESPN, Google, KSA, BGC
  - Generic terms: Affiliate, Awards, Commission, Court

**Combined approach:**
```python
def should_ignore(entity_name: str) -> bool:
    # Check combined ignore lists
    if entity_name in ALL_IGNORE_ENTITIES:
        return True

    # Use taxonomy logic if available
    if IMPORTED_TAXONOMY and taxonomy_should_ignore(entity_name):
        return True

    # Basic filters
    if len(entity_name) < 3 or entity_name.isdigit():
        return True

    return False
```

### 5. Context Extraction

For each company mention, extracts a context snippet:

**Algorithm:**
```python
def extract_context_snippet(text: str, entity_text: str, max_length: int = 240):
    1. Find entity position in text
    2. Scan backwards to find sentence start (. ! ?)
    3. Scan forwards to find sentence end
    4. Extract full sentence
    5. Truncate to max_length if needed (at word boundary)
```

**Example:**
- Input: "LeoVegas launches online sports betting in Germany..."
- Entity: "LeoVegas"
- Output: "LeoVegas launches online sports betting in Germany - igamingbusiness."

### 6. Aggregation

For each canonical company name:
- Count total mentions across all articles
- Collect all raw name variants
- Store up to 10 unique context snippets
- Track article IDs to avoid duplicate snippets

### 7. Filtering

Only includes companies with **≥ 3 mentions** to reduce noise.

|-------|------|-------------|

## Troubleshooting

### Issue: Too Few Companies Extracted

**Causes:**
- Aggressive noise filtering
- High minimum mentions threshold
- Small news history dataset

**Solutions:**
1. Lower `min_mentions` threshold (e.g., 2 instead of 3)
2. Review and reduce `ADDITIONAL_IGNORE_ENTITIES`
3. Run `main.py` multiple times to build larger history

### Issue: Too Many Noise Entities

**Causes:**
- Insufficient filtering
- spaCy NER false positives

**Solutions:**
1. Add problematic entities to `ADDITIONAL_IGNORE_ENTITIES`
2. Increase `min_mentions` threshold (e.g., 5 instead of 3)
3. Review context snippets to identify patterns

### Issue: Missing Known Companies

**Causes:**
- Company mentioned with uncommon variant
- Company filtered as noise
- Below minimum mentions threshold

**Solutions:**
1. Check raw NER output (print all ORG entities)
2. Verify company not in ignore lists
3. Lower minimum threshold temporarily

### Issue: Poor Context Snippets

**Causes:**
- Sentence boundary detection failing
- Truncation cutting important words

**Solutions:**
1. Adjust `max_length` parameter
2. Improve sentence boundary regex
3. Use full title/summary instead of just sentence
